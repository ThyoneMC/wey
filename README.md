---

> wéy (Indo-European) means "we"

## Setup

```bash
wey config --key=curseforgeApi
```

## Install Mods

1st Method: **Local Create**

```bash
wey create
wey add modrinth|curseforge
wey load
```

2nd Method: **Import**

```bash
wey import
wey update
wey load
```
