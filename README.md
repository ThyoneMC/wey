# Config

> Path: `/%appdata%/.thyonemc/wey/config.json`

Tunnel (Port forwarding)
```json
{
  "tunnel": {
		ngrok: "/path/to/ngrok",
		playit_gg: "/usr/bin/playit",
		hamachi: "/x64/hamchi.exe"
	}
}
```

Minecraft Server Port (default: 25565)
```json
{
  "auto_restart_delay": 25565
}
```

Minecraft Server Port (default: 25565)

{
  "port": 25565
}
